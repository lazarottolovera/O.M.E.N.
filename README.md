# O.M.E.N. (Orbital Monitoring & Encounter Network)

This is the official repository for the O.M.E.N. infrastructure.
Developed to consolidate historical UAP evidence and provide logistical/analytical support in the event of systemic instability (Protocol PURSUE).

### Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel (Edge-ready)
- **Data:** Real-time integration with monitoring APIs.

*The truth does not need permission to be processed.*
